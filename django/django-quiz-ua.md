#### П1. Щоб кешувати весь сайт для додатка в Django, ви додаєте всі налаштування, крім якого з цих?

- [ ] django.middleware.common.CommonMiddleware
- [ ] django.middleware.cache.UpdateCacheMiddleware
- [ ] django.middleware.cache.FetchFromCacheMiddleware
- [x] django.middleware.cache.AcceleratedCacheMiddleware

**Посилання:**
Django постачається з надійною системою кешування, яка дозволяє зберігати динамічні сторінки, щоб їх не потрібно було обчислювати для кожного запиту. Для зручності Django пропонує кешування з різною деталізацією — від цілого веб-сайту до сторінок, частин сторінок, результатів запитів до БД та будь-яких об'єктів у пам'яті. Проміжне програмне забезпечення кешу. Якщо ввімкнено, кожна сторінка на базі Django буде кешуватися на основі URL.
